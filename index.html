Fun with Filters

Processes used:

  1. Removes background with color selecting tool and eraser.
  2. Add solid color background.
  3. Draw stars with Photoshop's star brush.
  4. Draw hearts with a fuzzy brush.
  5. Use photoshops pre-defined filters to change brush strokes and textures.

Selective Black & White

Processes used:

  1. Create a duplicate layer.
  2. Add a layer mask to the layer on top.
  3. Make the t-shirt and fire extinguisher transparent on top layer using layer mask.
  4. Make top layer black and white through desaturation.
  5. Mess with brightness and contrast.

Warhol

Processes used:

  1. Remove background using pen tool.
  2. Stylize portrait with Cutout filter.
  3. Tile the portrait into 4 sections.
  4. Add a color background and portrait color and combine them using screen blending.
  5. Mess with brightness and contrast.
